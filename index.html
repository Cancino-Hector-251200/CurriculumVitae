Proyectos

CEIL

CEIL es una aplicación móvil diseñada para ayudar a estudiantes universitarios a mejorar su conocimiento financiero mediante herramientas simples, dinámicas e interactivas. La aplicación busca enseñar educación financiera de forma práctica utilizando retos, alertas inteligentes, control de gastos y elementos de gamificación para motivar el ahorro y el uso responsable del dinero. (Spring Boot, AWS, Java, Android nativo con Jetpack Compose).

ClinicApp

Esta aplicación web permite a los pacientes agendar citas con doctores disponibles en la plataforma según sus horarios publicados. Al agendar, el doctor recibe una notificación y, al aceptar, el paciente es notificado. El paciente tiene un apartado para ver el estado de su cita, incluyendo notificaciones si llega a ser rechazada.

Upsecitos Market

Marketplace online para la venta de productos de calidad dentro de una comunidad universitaria, desarrollado especialmente para las necesidades de los estudiantes que tienen un negocio, esta aplicación busca facilitar la compra y venta de productos de manera eficiente y segura. fue desarrollada con tecnologías como Spring Boot, AWS y MySQL para el backend, y JavaScript, HTML y CSS para el frontend. La aplicación permite a los usuarios registrarse, crear perfiles de vendedor, listar productos, realizar compras y gestionar pedidos de manera sencilla.
